Utility library for Material Components for the Web.

Material Components Kit provides a set of Bootstrap-like utility classes for MDC Web to simplify front-end development.

Useful Material Design components.

Material Components Kit complements MDC Web with components which are not presented in the parent library.

Unopinionated about your JavaScript framework.

Material Components Kit is an SCSS library so it doesn't provide any JavaScript. It's aimed to work with any JavaScript wrapper for MDC Web.
